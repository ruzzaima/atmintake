MonthlySchedule

Overview

Allows you to set up a schedule task which run on monthly basis, this schedule also allows you to fully customized the behaviour in which the schedules will run.

Properties

Property Description
Day - The day of the month for the task to run
Hour - The hour of the day for the task to run
Minute - The minute of the hour for the task to run
IsJanuary - Select to run it in January
IsFebruary - Select to run it in February
IsMarch - Select to run it in March
IsApril - Select to run it in April
IsMay - Select to run it in May
IsJune - Select to run it in June
IsJuly - Select to run it in July
IsAugust - Select to run it in August
IsSeptember - Select to run it in September
IsOctober - Select to run it in October
IsNovember - Select to run it in November
IsDecember - Select to run it in December
IsLastDay - If this is set, it will run the last day of the month, regardless of the number of days for that month, this could be on 28th or 29th for Feb, and 30th for April, 31st of May and so on
Days - Run it on fixed days of every selected months
