CreateEntityActivity

Overview

Allow you to add new item to your custom Entity.

Properties

Property Description
EntityType - The name of the EntityDefinition
ReturnValuePath - Int32 Where the {Entity}Id is stored once it's been inserted
PropertyMappingCollection - The list of mapping from the Workflow variables to the Entity members
