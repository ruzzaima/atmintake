ReceiveActivity

Receive Activity is an asyn activity that will instantiate and will sit waiting for the actual event to happen it's like ScreenActivity' without the user interface element.

While ScreenActivity is meant for human task interaction, ReceiveActivity is meant for integration event at the API level only

Properties

Property Description
ArgumentPath The path to the variable where the received message will be stored. The message type received should be convertible to the message type specified in the variable
