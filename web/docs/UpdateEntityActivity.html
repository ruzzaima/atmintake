UpdateEntityActivity

Overview

Lets you update an entity in your application

Properties

Property Description
EntityType - The name of the Entity
EntityIdPath - The path to the workflow variable where the id to the entity is kept
UseVariable - Reserved for future use
IsUsingVariable - Reserved
PropertyMappingCollection - Mapping from the source workflow variables to the properties of the entity
