Mapping Activity

MappingActivity allows you to create a mapping task with your WorkflowDefinition. This will basicall execute TransformAsync method in your TransformDefinition.

Properties

Property Description
TransformDefinition - The name of your TransformDefinition object
InputPath - The variable name that will be the input to your transformation
OutputPath - The variable name where the output of the transformation will be stored

Remarks

This will generally will create a piece of code in your Workflow thats intantiate the TransformDefinition and invoke the TransformAsyn method.

Give you WorkflowDefinition has 2 Variable named

  • Input
  • Output

it will create a code like

var map = new MyTransformDefinition();
this.Output = await map.TransformAsync(this.Input);

as such your Input and Output must be of the same type specifed in the TransformDefinition Source and Destination
