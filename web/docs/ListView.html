ListView

Overview

Provides a tabular view of your collection, ListView is one of the container FormElement in which it could contains other elements. The main properties of the ListView is the ListViewColumnCollection which represents individual column in the table.

Properties

Property Description
ChildItemType - Full name for the child item type, this is normallu in form bespoke.<applicationName>_<EntityDefinitionId>.domain.<ChildItem> for entity form. While for ScreenActivity in workflow it's in the form bespoke.w_<WorkflowDefinitionId>_<version>.sph.<ChildItemType> when the child item is generated from XML Schema
IsChildItemFunction - (Obsolete)
ListViewColumnCollection - Columns for the listview see ListViewColum for details
