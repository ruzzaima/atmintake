DailySchedule

Overview

Allows you to set up a schedule task which run on daily basis, this schedule also allows you to fully customized the behaviour in which the schedules will run.

Properties

Property Description
Recur - How many days before it recurs
StartHour - The hour of the day for the task to run
StartMinute - The minute of the hour for the task to run
StartSecond - The seconds of the minute for the task to run
EndHour - The hour of the day for the task to stop
EndMinute - The minute of the hour for the task to stop
EndSecond - The seconds of the minute for the task to stop
