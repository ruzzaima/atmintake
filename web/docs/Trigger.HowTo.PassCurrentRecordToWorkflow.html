How to pass trigger current record to a StartWorkflowAction

Your WorkflowDefinition might contains a ClrTypeVariable which normally contains one of the EntityDefinition. It's common situation where you want to pass the current record from your Trigger to the workflow in order to start it. The StartWorkflowAction contains MappingCollection which allows you to map the WorkflowDefinition variables.

Lets pick an example, where your WorkflowDefinition contains a variable called Patient and is of type ClrTypeVariable. From your Trigger which point to the Patient EntityDefinition, The current item in your FunctionField is always the record.

So all you have to do is just write a FunctionField with one line of script

item

or

return item;
