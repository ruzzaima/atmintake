StartWorkflowAction

Overview

The most versatile of actions , in which it allows you to start a new WorkflowDefinition.

Properties

Property Description
WorkflowDefinitionId - Selected WorkflowDefinition Id
Name - The name of the action
Version - The workflow version
WorkflowTriggerMapCollection - Setting the workflow variable values
UseAsync - Internal use
